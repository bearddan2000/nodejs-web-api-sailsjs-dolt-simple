# nodejs-web-api-sailsjs-dolt-simple

## Description
A sailsjs api that queries dolt server.

## Tech stack
- sailsjs
- dolt

## Docker stack
- node:latest
- dolthub/dolt-sql-server

## To run
`sudo ./install.sh -u`
curl -i localhost/dogs

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
